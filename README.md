# proyecto-videojuegos
Este proyecto utiliza variables para guardar diccionarios y acceder a estos mediante claves(nombre de los juegos), las tuplas para asignar valores binarios de ventas y stocks para cada juego, los conjuntos para asignarle a cada juego los clientes que los compraron y funciones para mostrar la información de cada juego.

Se aprendió a utilizar las palabras claves para poder asignarlo lo datos de forma ordenada y segura, de esta reducimos el código y lo hacemos fácil de leer, la función se creó en base a los diccionarios que creamos y la lista con los videojuegos para poder crear un bucle en el que se imprima toda la información de los juegos utilizando la función resumen_juego.

Se creó otra función llamada mostrar_resumenes() para que muestre la información de los juegos que tengan 500 ventas o más, utilizando la función lambda creamos una función pequeña que nos permita sumar las ventas de todos los juegos que contengan más de 500.

El emplear el uso de lambda fue algo que no comprendía del todo pero resultó ser muy sencillo, solo nos permite crear una función pequeña de manera rápida para llamarle después.
